The dotbox-dropdown component provides a versatile dropdown/select with support for single and multiple selection, searchable options, custom tags, and more.

Examples

Basic Usage

<dotbox-dropdown label="Select a fruit" options='["Apple", "Banana", "Orange", "Mango"]'> </dotbox-dropdown> <dotbox-dropdown label="Select a country" options='[{"value": "us", "label": "United States"}, {"value": "ca", "label": "Canada"}, {"value": "mx", "label": "Mexico"}]'> </dotbox-dropdown>

Label Positions

<dotbox-dropdown label="Label Above" labelPosition="above" options='["Option 1", "Option 2", "Option 3"]'> </dotbox-dropdown> <dotbox-dropdown label="Label Before" labelPosition="before" options='["Option 1", "Option 2", "Option 3"]'> </dotbox-dropdown> <dotbox-dropdown label="A longer label before the dropdown" labelPosition="before" options='["Option 1", "Option 2", "Option 3"]'> </dotbox-dropdown>

Searchable Dropdown

<dotbox-dropdown label="Search countries" searchable options='["United States", "Canada", "Mexico", "United Kingdom", "France", "Germany", "Italy", "Spain", "Portugal", "Brazil", "Argentina", "Chile", "Australia", "New Zealand", "Japan", "China", "India"]'> </dotbox-dropdown>

Multiple Selection

<dotbox-dropdown label="Select multiple fruits" multiple options='["Apple", "Banana", "Orange", "Mango", "Pineapple", "Strawberry", "Blueberry"]'> </dotbox-dropdown>

Multiple Selection with Checkboxes

<dotbox-dropdown label="Select multiple items" multiple useCheckboxes options='["Item 1", "Item 2", "Item 3", "Item 4", "Item 5"]'> </dotbox-dropdown>

Multiple Selection with Tags

<dotbox-dropdown label="Select tags" multiple useTags options='["JavaScript", "HTML", "CSS", "React", "Vue", "Angular", "Node.js", "Python", "Java", "C#"]'> </dotbox-dropdown>

Searchable Multiple Selection with Tags

<dotbox-dropdown label="Search and select multiple tags" multiple searchable useTags options='["JavaScript", "HTML", "CSS", "React", "Vue", "Angular", "Node.js", "Python", "Java", "C#", "PHP", "Ruby", "Swift", "Kotlin", "Go", "Rust", "TypeScript", "Scala", "Perl", "Haskell"]'> </dotbox-dropdown>
