API Reference

Properties

Name Type Default Description
variant String "default" Card variant (primary, success, danger, info, default)
title String "" Card title
icon String "" FontAwesome icon name for the card header (e.g., 'fa-info-circle')
hoverEffect Boolean false Whether to show a shadow effect on hover

CSS Parts

Name Description
card The card element
header The card header
body The card body
footer The card footer

Slots

Name Description
default Default slot for card content
header Slot for custom header content
footer Slot for footer content
