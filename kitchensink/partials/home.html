Getting Started

To use Dotbox Components in your project:

  1. Install the package: npm install dotbox-components
  2. Import the components: import 'dotbox-components';
  3. Use the components in your HTML: <dotbox-button>Click Me</dotbox-button>

CSS Bundling for Production

For production use, you can include the bundled CSS file:

<link rel="stylesheet" href="node_modules/dotbox-components/dist/dotbox-components.bundled.css">

Or for more granular control, include CSS for specific components:

<link rel="stylesheet" href="node_modules/dotbox-components/dist/css/base/base-styles.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/button/button.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/card/card.css">
