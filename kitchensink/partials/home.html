Dotbox Components

A reusable web component library with stylish buttons and cards.

Explore Components

Getting Started

To use Dotbox Components in your project:

  1. Install the package: npm install dotbox-components
  2. Import the components: import 'dotbox-components';
  3. Use the components in your HTML: <dotbox-button variant="primary">Click Me</dotbox-button>
Tip: Check out the individual component pages for detailed usage examples.

CSS Bundling for Production
Option 1: Include the bundled CSS file

For production use, you can include the bundled CSS file:

<link rel="stylesheet" href="node_modules/dotbox-components/dist/dotbox-components.bundled.css">
Option 2: Include CSS for specific components

For more granular control, include CSS for specific components:

<link rel="stylesheet" href="node_modules/dotbox-components/dist/css/base/base-styles.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/button/button.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/card/card.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/checkbox/checkbox.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/dialog/dialog.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/input/input.css"> <link rel="stylesheet" href="node_modules/dotbox-components/dist/css/notification/notification.css">
Note: Make sure to include the base-styles.css file when using individual component CSS files.
