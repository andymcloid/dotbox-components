The Checkbox component provides a versatile way to create checkboxes, radio buttons, and toggle switches with consistent styling and behavior.

Basic Usage

Regular Checkboxes
<dotbox-checkbox label="Default checkbox"></dotbox-checkbox> <dotbox-checkbox label="Checked checkbox" checked></dotbox-checkbox> <dotbox-checkbox label="Disabled checkbox" disabled></dotbox-checkbox> <dotbox-checkbox label="Disabled checked checkbox" checked disabled></dotbox-checkbox>
Radio Buttons
<dotbox-checkbox type="radio" name="radio-group" label="Radio option 1" value="option1"></dotbox-checkbox> <dotbox-checkbox type="radio" name="radio-group" label="Radio option 2" value="option2" checked></dotbox-checkbox> <dotbox-checkbox type="radio" name="radio-group" label="Radio option 3" value="option3"></dotbox-checkbox> <dotbox-checkbox type="radio" name="radio-group" label="Disabled radio option" value="option4" disabled></dotbox-checkbox>
Toggle Switches
<dotbox-checkbox type="toggle" label="Default toggle"></dotbox-checkbox> <dotbox-checkbox type="toggle" label="Checked toggle" checked></dotbox-checkbox> <dotbox-checkbox type="toggle" label="Disabled toggle" disabled></dotbox-checkbox> <dotbox-checkbox type="toggle" label="Disabled checked toggle" checked disabled></dotbox-checkbox>
