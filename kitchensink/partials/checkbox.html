Checkbox Component

The Checkbox component provides a versatile way to create checkboxes, radio buttons, and toggle switches with consistent styling and behavior.

Basic Usage

Regular Checkboxes
<dotbox-checkbox label="Default checkbox"></dotbox-checkbox>
<dotbox-checkbox label="Checked checkbox" checked></dotbox-checkbox>
<dotbox-checkbox label="Disabled checkbox" disabled></dotbox-checkbox>
<dotbox-checkbox label="Disabled checked checkbox" checked disabled></dotbox-checkbox>
Radio Buttons
<dotbox-checkbox type="radio" name="radio-group" label="Radio option 1" value="option1"></dotbox-checkbox>
<dotbox-checkbox type="radio" name="radio-group" label="Radio option 2" value="option2" checked></dotbox-checkbox>
<dotbox-checkbox type="radio" name="radio-group" label="Radio option 3" value="option3"></dotbox-checkbox>
<dotbox-checkbox type="radio" name="radio-group" label="Disabled radio option" value="option4" disabled></dotbox-checkbox>
Toggle Switches
<dotbox-checkbox type="toggle" label="Default toggle"></dotbox-checkbox>
<dotbox-checkbox type="toggle" label="Checked toggle" checked></dotbox-checkbox>
<dotbox-checkbox type="toggle" label="Disabled toggle" disabled></dotbox-checkbox>
<dotbox-checkbox type="toggle" label="Disabled checked toggle" checked disabled></dotbox-checkbox>

Event Handling

Change Event
Change event will be shown here 
<dotbox-checkbox id="event-demo-checkbox" label="Click me"></dotbox-checkbox>
<div id="event-demo-output" class="alert alert-info">
    Change event will be shown here
</div>

<script>
    document.getElementById('event-demo-checkbox').addEventListener('change', (e) => {
        document.getElementById('event-demo-output').textContent = 
            `Change event: checked = ${e.detail.checked}, value = ${e.detail.value}`;
    });
</script>

API Reference

Properties
Name Type Default Description
type String "checkbox" Type of input: "checkbox", "radio", or "toggle"
name String "" Name attribute for the input
value String "" Value attribute for the input
checked Boolean false Whether the input is checked
disabled Boolean false Whether the input is disabled
variant String "primary" Visual variant: "primary", "success", "danger", "info", "default"
size String "" Size of the input: "small", "", "large"
label String "" Label text for the input
labelPosition String "right" Position of the label: "left" or "right"
required Boolean false Whether the input is required
Events
Name Detail Description
change { checked: Boolean, value: String } Dispatched when the input value changes
input { checked: Boolean, value: String } Dispatched when the input value is being changed
CSS Parts
Name Description
container The container element
input The input element
label The label element
toggle The toggle switch element (only for toggle type)
slider The slider element (only for toggle type)
