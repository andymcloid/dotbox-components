API Reference

Properties

Name Type Default Description
variant String "default" Dialog variant (primary, success, danger, info, default)
title String "" Dialog title
icon String "" FontAwesome icon name for the dialog header (e.g., 'fa-info-circle')
open Boolean false Whether the dialog is open
modal Boolean true Whether the dialog is modal (shows overlay and prevents background interaction)
size String "medium" Dialog size (small, medium, large)

Methods

Name Parameters Description
show() None Opens the dialog
close() None Closes the dialog
toggle() None Toggles the dialog between open and closed states

Events

Name Description
open Fired when the dialog is opened
close Fired when the dialog is closed

CSS Parts

Name Description
dialog The dialog element
overlay The overlay element (when modal)
header The dialog header
body The dialog body
footer The dialog footer
close-button The close button

Slots

Name Description
default Default slot for dialog content
header Slot for custom header content
footer Slot for footer content
