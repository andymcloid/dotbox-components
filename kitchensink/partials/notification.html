The dotbox-notification component provides stylish notifications with various options for variants, positions, and stacking behavior.

Examples

Basic Usage

Default Notification Success Notification Warning Notification Danger Notification Info Notification
<script> // Show a default notification DotboxNotification.show({ title: 'Default Notification', message: 'This is a default notification', variant: 'default' }); // Show a success notification DotboxNotification.show({ title: 'Success!', message: 'Operation completed successfully', variant: 'success' }); // Show a warning notification DotboxNotification.show({ title: 'Warning', message: 'Please check your input', variant: 'warning' }); // Show a danger notification DotboxNotification.show({ title: 'Error', message: 'Something went wrong', variant: 'danger' }); // Show an info notification DotboxNotification.show({ title: 'Information', message: 'Here is some information', variant: 'info' }); </script>

Positions

Top Right Top Left Bottom Right Bottom Left Top Center Bottom Center
<script> // Show notification at top-right (default) DotboxNotification.show({ title: 'Top Right', message: 'This notification appears at the top-right', position: 'top-right' }); // Show notification at top-left DotboxNotification.show({ title: 'Top Left', message: 'This notification appears at the top-left', position: 'top-left' }); // Show notification at bottom-right DotboxNotification.show({ title: 'Bottom Right', message: 'This notification appears at the bottom-right', position: 'bottom-right' }); // Show notification at bottom-left DotboxNotification.show({ title: 'Bottom Left', message: 'This notification appears at the bottom-left', position: 'bottom-left' }); // Show notification at top-center DotboxNotification.show({ title: 'Top Center', message: 'This notification appears at the top-center', position: 'top-center' }); // Show notification at bottom-center DotboxNotification.show({ title: 'Bottom Center', message: 'This notification appears at the bottom-center', position: 'bottom-center' }); </script>

Stacking Behavior

Show Stacking Notifications Show Non-Stacking Notifications
<script> // Show stacking notifications function showStackingNotifications() { DotboxNotification.show({ title: 'First Notification', message: 'This is the first notification', variant: 'info', stacking: true }); setTimeout(() => { DotboxNotification.show({ title: 'Second Notification', message: 'This is the second notification', variant: 'success', stacking: true }); }, 500); setTimeout(() => { DotboxNotification.show({ title: 'Third Notification', message: 'This is the third notification', variant: 'warning', stacking: true }); }, 1000); } // Show non-stacking notifications function showNonStackingNotifications() { DotboxNotification.show({ title: 'First Notification', message: 'This will be replaced', variant: 'info', stacking: false }); setTimeout(() => { DotboxNotification.show({ title: 'Second Notification', message: 'This replaces the first one', variant: 'success', stacking: false }); }, 1000); setTimeout(() => { DotboxNotification.show({ title: 'Third Notification', message: 'This replaces the second one', variant: 'warning', stacking: false }); }, 2000); } </script>

Duration

Auto-close (5s) No Auto-close
<script> // Show notification with auto-close DotboxNotification.show({ title: 'Auto-close', message: 'This notification will close after 5 seconds', duration: 5000 // 5 seconds }); // Show notification without auto-close DotboxNotification.show({ title: 'No Auto-close', message: 'This notification will stay until manually closed', duration: 0 // No auto-close }); </script>

With Icons

Custom Icon No Icon
<script> // Show notification with custom icon DotboxNotification.show({ title: 'Custom Icon', message: 'This notification has a custom icon', icon: 'fa-star' }); // Show notification without icon DotboxNotification.show({ title: 'No Icon', message: 'This notification has no icon', icon: '' // Empty string to override default icon }); </script>
